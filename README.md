## Setup

```
$ yarn install
```

## Build

Build for staging:

```
$ yarn build
```

Build for production:

```
$ yarn build-prod
```

## Development

```
$ REACT_APP_BACKEND_URL=https://dev.eyeseetea.com/play yarn start
```

## Tests

```
$ yarn test
```

Run integration tests locally:

```
$ yarn cy:open # interactive UI
$ yarn cy:run # non-interactive UI
```
